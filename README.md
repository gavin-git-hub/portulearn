# PortuLearn 🇧🇷

> 通过 YouTube 视频学习巴西葡萄牙语的一站式 Web 工具

## ✨ 最新功能特性

- 🎬 **YouTube 视频嵌入** — 输入任意葡语 YouTube 链接，自动记忆上次播放
- 📖 **故事分段循环** — 自动按分钟分割视频，也可自定义起止时间，精准跳转
- 🔁 **段内循环播放** — 在当前故事段内无限循环，强化听力
- 🗣 **双语字幕同步** — 自动获取 YouTube 内置字幕，支持葡语 + 中文实时高亮
- 📂 **本地字幕导入** — 支持导入 `.srt` 或 `.vtt` 文件，强行完美对齐你的时间轴
- 🤖 **AI 词汇分析** — 接入 **Gemini** 和 **DeepSeek**，根据双语字幕智能提取词汇、生成对照
- 💾 **字幕缓存秒开** — 自动持久化保存翻译和导入的字幕，下次打开零流量秒开
- 📱 **完美适配手机** — 支持桌面与手机自适应排版，将网页添加到桌面即可作为原生 App 使用
- 🃏 **翻转抽认卡** — 互动卡片练习记忆
- 📥 **学习材料导出** — 一键导出 Markdown 格式

## 🚀 如何打开使用

你有三种方式可以打开并使用 PortuLearn：

### 方法一：直接访问线上网址（最推荐）
无需任何配置，在电脑或手机的浏览器中直接打开即可（支持保存为手机 App 图标）：
👉 **[https://gavin-git-hub.github.io/portulearn/](https://gavin-git-hub.github.io/portulearn/)**

### 方法二：使用命令行（Mac / Linux 一键脚本）
如果你把代码下载到了本地，并且是 Mac 或 Linux 用户：
```bash
chmod +x start.sh && ./start.sh
```
执行后会自动在浏览器中打开 http://localhost:8765

### 方法三：手动启动本地服务器（Windows / 其他）
如果你在本地环境，可以使用 Python 自带的 HTTP 服务器启动：
```bash
python3 -m http.server 8765
```
然后在浏览器中输入网址：[http://localhost:8765](http://localhost:8765)

> ⚠️ **注意**：由于浏览器的安全限制，**不能**直接双击 `index.html` (使用 `file://` 协议) 打开，否则 YouTube API 会报错（Error 153）且无法保存你的设置，必须通过上述任意一种 `http(s)` 的方式访问。

## 💡 使用指南

1. **输入 YouTube 链接** → 点击「▶ 开始」
2. **输入 API Key**（在设置 ⚙️ 中可切换使用 Gemini 或 DeepSeek 的 Key）
3. **获取字幕**：
   - 默认尝试抓取视频自带的葡语字幕。
   - 点击「🌐 翻译字幕」可让 AI 自动补充中文翻译。
   - 点击「📂 导入字幕」可使用自己下载的 `.srt` 精准字幕覆盖。
4. **视频分段**：自动分为 5 分钟一段（或在设置中自定义），默认循环当前段落。
5. **AI 分析**：点击「✨ AI分析」，即可让 AI 结合当前句子的中葡双语进行深度拆解分析。
6. **导出**：点击「📚 学习材料」查看、复习并导出所有学习记录。

## ⌨️ 快捷键

| 键 | 功能 |
|----|------|
| `空格` | 播放 / 暂停 |
| `← →` | 后退 / 前进 5 秒 |
| `↑ ↓` | 上一段 / 下一段 |
| `R` | 重播当前段 |
| `L` | 切换循环 |

## 🛠️ 技术栈

- **纯 HTML/CSS/JS** — 无框架依赖，单文件极简部署
- **YouTube IFrame API** — 视频嵌入与控制
- **Google Generative AI / DeepSeek API** — 支持最新大模型，词汇分析与翻译
- **localStorage** — 浏览器本地持久化保存配置、字幕、和学习材料
